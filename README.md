# security-learning-portal

Portal central de la plataforma de tutoriales de seguridad.

Este portal reúne el catálogo de cursos, su versión, su nivel de dificultad, el tiempo estimado y los enlaces para crear cada repositorio desde template.

## Objetivos

- descubrir cursos por temática
- entrar rápidamente en un tutorial
- ver el estado de cada template
- seguir la evolución de versiones
- acceder al contenido completado y a las medallas finales

## Botones de acceso

[Volver al catálogo](./)

[Ver repositorios](./catalog/repositories.yml)

[Ver versiones](./catalog/versions.json)

[Crear un repositorio desde template](https://github.com/new)

## Estructura del portal

- `catalog/repositories.yml`: inventario de repositorios y metadatos
- `catalog/versions.json`: estado de versión de cada template
- documentación del roadmap y del mantenimiento

## Estado

Este portal es el índice de toda la plataforma. Cada tutorial individual enlaza de vuelta aquí para mantener navegación consistente entre cursos.
